
JS3tream is a java application.  To run this program, you must execute it in the directory that JS3tream was
installed. This is because JS3tream expects to find all of it's jar files in the current working
directory.  


To run JS3tream.. execute
> java -jar js3tream.jar  <options>


You can get a list of options with
> java -jar js3tream.jar -h

For more help see the main web site
http://js3tream.sourceforge.net
